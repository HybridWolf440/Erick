JavaScript Introducción

JavaScript o también conocido como JS es un lenguaje de programación orientado a objetos, dialecto del estándar ECMAScript.

JS fue desarollado originalmente por Berndan Eich de Netscape con el nombre de Mocha,Después fue nombrado como LifeScript y finalmente a día de hoy JavaScript

El cambio de nombre coincidió con el momento en que netscape agregó compatibilidad con la tecnología Java en su navegador en diciembre de 1995

Este cambio de nombre dio la impresion de que el lenguaje es una prolongacion de Java. Pero esto no es así ya que tienen diferentes semánticas y propósitos En 1997 los autores propusieron a Js para que fuera el estándar de la European Computer Manufacturers 'Association ECMA. En junio de 1997 fue adoptado como el estándar ECMa, con el nombre de ECMAScript. Poco tiempo desués también como estándar ISO.JS se diseñó con una sintaxis similar a C, aunque usa nombres y convenciones del lenguaje Java

Se usa principalmente del lado del cliente, de manera que es parte de un navegador web permitiendo así mejoras en la interfaz de usuario y páginas web dinámicas, aunque existe una forma de JS del lado del sevidos (Server-side JavaScript o SSJS). En 2012, todos los navegadores modernos soportan ECMAScript 5.1, una version de JS. Los más antiguos al menos soportan ECMAScript 3.

Insertar JavaScript en HTML

Hay dos maneras de hacerlo

  • Agrega JavaScript directamente en el archivo HTMl

    • El codigo puede ser colocado tanto en el head como en el body.

    • Head: La carga será diferente. Se recomienda agregarlo en el head para que este separado del HTML.
    • Body: Ayuda a la que la carga sea más rapida, ya que el contenido cargara más rapido y después de eso se procesa más rapido el JavaScript
  • Agrega codigo JavaScript en un archivo separado

Método - getElementbyId()

El método getElementbyId() devuelve el elemento que tiene el atributo ID con el valor que es especificado.

Este es uno de los métodos más comunes en el DOM de HTML y se usa cada que se desea manipular u obtener información de un elemento del documento

Si no existen elenentos con el Id que es especificado devuelve nulo

Un ID debe ser unico dentro de una página. Pero si existe más de un Id igual, el método getElementbyId() devuelve el primer elemento Id con ese nombre que encuentre

variables

Las variables de JavaScript son contendeores para almacenar valores de datos

todas las variables de JavaScript deben identificarse con nombres unicos. Estos nombres unicos son llamados identificadores

Los identificadores pueden llevar nombres cortos (X, Y) o nombres más descriptivos (edad, suma, total, nombre)

Se usa "var, let y/o const" para crear variables

var es la abreviatura de variable, es una palabra clave de JavaScript que crea, o declara una nueva variable.

Reglas generales para construir nombres para variables

  • Los nombres deben comenzar con una letra.
  • Los nombres pueden contener letras, dígitos, guiones bajos y signos de dólar.
  • Los nombres también pueden comenzar con $ y _
  • Los nombres distinguen entre mayúsculas y minúsculas (N y n son variables diferentes)
  • Las palabras reservadas (Como las palavras clave de JavaScript no se pueden usar como nombres)

Operadores aritméticos

JavaScript tiene un amplio conjunto de operadores, en los que se incluyen los famosos operadores aritméticos, logicos, bit a bit y de asignación, así como algunos operadores varios.

Los operadores permiten realizar cpalculos complejos y tomar decisiones lógicas en función de comparaciones y otros tipos de condiciones

Los operadores aritméticos estándar son adición o suma (+), sustracción o resta(-), multiplicación(*), divicion(/), incremento(++), decremento(--)

Tipos de datos

Cadena (string)

un valor de cadena esta formado por una cadena de cero o más caracteres Unicode (letras, dígitos y signos de puntuación). Este tipo de datos se usa para reprecentar texto en JavaScript

Número

En JavaScript, no se distingue entre valores enteros y de punto flotante; un número de JavaScript puede ser cualquiera de ellos (Internamente, JavaScript representa todos los números como valores de punto flotante)

Los valores enteros pueden ser números enteros positivos, númerso enteros negativos y 0.

Boleano

Este tipo de datos pueden tener unicamente dos valores diferentes, los cuales son: True y false

Un valor booleano es u valor de validez: especifica si una condición es verdadera o no

Eventos

Un evento HTML puede ser algo que hace el navegador, o algo que hace un usuario. JavaScript puede "reaccionar" a esto eventos

Algunos comunes son:

  • onchange: Un elemento HTML ha sido cambiado
  • onclick: El usuario hace click en un elemento HTML
  • onmouseover: El usuario mueve el mouse sobre un elemento HTML
  • onmouseout: El usuario aleja el mouse del elemento HTML
  • onkeydown: El usaurio presiona una tecla del teclado
  • onload: El navegador ha terminado de cargar la página
